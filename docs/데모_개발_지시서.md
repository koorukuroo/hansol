---

## 프로젝트 개요

- **목적**: 정부 사업 기술성평가 발표(20분) 시 라이브 시연할 웹 데모
- **핵심**: 덕양가스(산업용 가스 물류) 실측 데이터로 작동하는 AI 관제 플랫폼
- **발표일**: 2026년 5월 중 (정확한 일정 추후 확정)
- **데모 시간**: 발표 20분 중 약 8~10분

### Page 2: 수요예측 VMI (`app/forecast/page.tsx`)

**좌측 패널 (w-[400px]):**
- 고객 탱크 잔량 리스트
- 각 행: 위험도 아이콘(🔴🟡🟢) + 거래처명 + 품목 + 잔량 게이지 바 + 고갈 예상일
- 위험도순 정렬 (danger → warning → safe)
- 행 클릭 시 우측 상세 패널 변경
- 검색 필터 (거래처명, 품목)

**우측 상세 패널 (flex-1):**
- 선택된 거래처 정보 카드
- 14일 예측 차트 (Recharts ComposedChart):
  - 파란 실선: 과거 실측값 (30일)
  - 주황 실선: AI 예측값 (14일)
  - 주황 반투명 영역: 95% 신뢰구간
  - 빨간 점선: 위험 임계선 (20%)
- 예측 결과 요약 카드:
  - 현재 잔량, 일평균 소비, 고갈 예상일, 권장 보충량
  - 주요 영향 변수 3개 (topFactors)
- **"VMI 자동보충 주문 생성" 버튼** (bg-blue-600):
  - 클릭 → 토스트 알림: "주문이 생성되었습니다. 배차최적화에 반영됩니다."
  - 1.5초 후 자동 소멸

### Page 3: 배차최적화 (`app/dispatch/page.tsx`)

**좌측: 배송 요청 리스트 (w-[350px])**
- 체크박스 + 거래처명 + 품목 + 수량
- VMI 자동주문과 카톡 수동주문 구분 (태그)
- 하단: **"🧠 AI 최적 배차 실행" 버튼** (bg-green-600, 큰 사이즈)
  - 클릭 → 2초 프로그레스 바 → 결과 표시

**우측 상단: 배차 결과 (flex-1)**
- 차량별 카드:
  - 차량번호 + 기사명
  - 방문 순서: 본사 → A → B → 본사
  - 총 거리, 배송 건수
- 카드 클릭 시 지도에 해당 경로 하이라이트

**우측 하단: 지도 (Before/After)**
- 탭 2개: [Before] [After]
- Before: 빨간/주황 경로선 (비효율적, 교차 많음)
- After: 초록/파란 경로선 (깔끔한 최적 경로)
- 탭 전환 시 경로 애니메이션 (Framer Motion)
- **비교 수치 표시:**
  - 총 거리: 1,240km → 890km (▼28%)
  - 공차율: 7.6% → 3.2% (▼58%)
  - 배차 시간: 40분 → 5분 (▼88%)

### Page 4: 카카오톡 발주 AI (`app/kakao/page.tsx`)

**좌측: 카카오톡 채팅 UI (w-[400px])**
- 카카오톡 스타일 (배경: #B2C7D9, 말풍선: 흰색 둥근 사각형)
- 샘플 메시지 5개 프리셋 (kakao_samples.json)
