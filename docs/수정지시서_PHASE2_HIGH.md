## 2-1. AI 기술 적용 계획 구체화 (서면 20점 + 기술성 15점)

### 삽입 위치: II.1 AI 응용제품 신속 상용화 계획 도입부 (p.6~7)

현재 4개 모듈의 기술 설명이 있으나, **모듈별 구현 비중**과 **"범용 AI가 아닌 제조 특화"** 강조가 부족.
아래 내용을 기술 설명 앞에 삽입:

```
□ AI 핵심기술 구성 및 구현 비중

  본 과제에서 도입하는 AI 핵심기술은 단순 범용 AI(기존 LLM, 범용 챗봇 등)가 아닌, 
  위험물 가스 물류 도메인에 특화된 제조 AI로서, 각 모듈별 구현 비중은 다음과 같다.

  ┌─────────────────────────────────────────────────────────────┐
  │ AI 모듈         │ 개발 유형           │ 비중 │ 핵심 기술        │
  ├─────────────────┼────────────────────┼──────┼────────────────┤
  │ ① 수요예측(VMI) │ 자체 개발           │ 40%  │ LSTM+Prophet    │
  │                 │                    │      │ 앙상블,          │
  │                 │                    │      │ XGBoost/LightGBM│
  ├─────────────────┼────────────────────┼──────┼────────────────┤
  │ ② 배차최적화    │ 자체 개발           │ 35%  │ OR-Tools CVRPTW,│
  │                 │                    │      │ ALNS, PPO 강화  │
  │                 │                    │      │ 학습             │
  ├─────────────────┼────────────────────┼──────┼────────────────┤
  │ ③ 물류 차량     │ 기성 AI + 커스터마  │ 15%  │ ATALAS AI 엔진, │
  │   안전 운행     │ 이징               │      │ YOLOv8 객체인식  │
  ├─────────────────┼────────────────────┼──────┼────────────────┤
  │ ④ 예지보전      │ 경량 ML + 룰엔진   │ 10%  │ LSTM 잔존수명,  │
  │                 │                    │      │ 이상탐지         │
  └─────────────────────────────────────────────────────────────┘

  ※ 자체 개발 비중 75% (수요예측 + 배차최적화) — 도메인 특화 알고리즘 직접 설계·구현
  ※ 기성 제품 통합 비중 25% (어라운드뷰 + 예지보전) — 검증된 AI 엔진 + 현장 맞춤 커스터마이징
  ※ 범용 LLM/ChatGPT 류 기술 활용 비중: 0% (전량 제조 특화 AI)

□ 제조 도메인 특화 요소

  (1) 수요예측 — 범용 시계열 예측이 아닌, 가스 소비 특화 모델
    - 산업용 가스 소비의 고유한 패턴: 평일/휴일 이원 구조, 기온 역비례 특성(난방용),
      산업단지 가동률 연동, 탱크 용량별 소비 곡선 차이
    - 외부 변수 융합: 기상청 예보 API + 통계청 산업가동률 + 고객사 조업 캘린더
    - 위험물안전관리법상 최소 잔량 유지 의무 반영 (안전 재고 제약)

  (2) 배차최적화 — 범용 VRP가 아닌, 위험물 물류 제약 반영 모델
    - 위험물안전관리법 제약: 혼적 금지 품목, 적재중량 상한, 운행시간 제한
    - 고압가스 충전소 경유 제약: 충전소 위치·운영시간·대기열 반영
    - 탱크 잔량 기반 우선순위: 고갈 임박 탱크 긴급 배송 동적 삽입
    - 강화학습(PPO) 기반 실시간 리라우팅: 교통 상황·돌발 주문 대응

  (3) 물류 차량 안전 운행 — 범용 ADAS가 아닌, 위험물 운송 차량 특화
    - 위험물 표지 차량의 사각지대 특성 반영 (탱크로리 후방/측면)
    - 산업단지 내 특수 환경 인식 (지게차, 배관, 밸브 스테이션 등)
    - 위험물 관련 이벤트 분류: 급제동/급가속 시 적재물 위험도 연동

  (4) 예지보전 — 범용 설비보전이 아닌, 운송 차량 특화
    - 위험물 운송 차량의 특수 부품: 탱크 밸브, 안전장치, 방폭 설비
    - 주행 패턴(장거리/단거리, 평지/산지)별 마모도 차등 예측
    - 법정 검사 일정과 연계한 보전 스케줄링
```

```
    (6) 저장: TimescaleDB(시계열), PostgreSQL(메타), S3(원본 아카이브)
```

## 2-4. ESG 전략 (기술성 5점)

### 삽입 위치: IV. 확산 및 성과활용 계획 내, 또는 별도 소절

```
□ ESG 기반 사회적 가치 창출 계획

  (1) 환경(E: Environment)
    ○ 배출가스 감축:
      - 배차최적화로 공차율 50% 감소 → 연간 약 OO만km 주행 절감
      - 절감 주행거리 × 배출계수(2.6kg CO2/L) = 연간 약 120톤 CO2 감축
      ※ 산출근거: 차량 60대 × 1일 평균 OOkm 절감 × 250영업일 × 0.12L/km × 2.6kg/L
    ○ 가스 누출 위험 감소:
      - IoT 실시간 모니터링으로 탱크 고갈 방지 → 긴급 방출/누출 사고 예방
      - 목표: 가스 관련 환경 사고 제로(0건)

  (2) 사회(S: Social)
    ○ 지역사회 안전 기여:
      - 위험물 운송 사고 감소: 어라운드뷰 AI로 운행 중 사고 60% 감소 목표
      - 울산/경남 산업단지 내 위험물 차량 안전도 향상
    ○ 고용 창출:
      - AI 데이터분석가 O명, 스마트물류 관제사 O명 신규 채용
      - 기존 배차담당자의 AI 활용 역량 전환 교육(디지털 리스킬링)
```
