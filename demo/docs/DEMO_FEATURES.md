# 덕양가스 AI 통합 관제 플랫폼 — 데모 기능 명세서

> 최종 업데이트: 2026-04-15
> 기술 스택: Next.js 16 + TypeScript + Tailwind v4 + Recharts + react-leaflet + Framer Motion

---

## 시스템 개요

덕양가스 울산 본사의 산업용 가스(N2, O2, CO2, AR, LPG) 물류를 AI로 관제하는 통합 플랫폼 데모입니다. 수요예측, 배차최적화, 텔레매틱스, 안전관제, 예지보전까지 물류 전 과정을 하나의 시스템에서 관리합니다.

**핵심 수치:**
- 거래처: 15개 (울산/경남 산업단지)
- 차량: 20대 (벌크 탱크로리 10대 + 실린더 운반차 10대)
- 기사: 20명
- 배송 기록: 7,627건 (실측 데이터 기반)

## 페이지별 기능 상세

### 1. 통합 대시보드 (`/`)

전체 운영 현황을 한 눈에 파악하는 메인 화면.

**상단 배너**
- 네이비 그래디언트 배경
- "위험 거래처 N곳 — OO 외 N곳 고갈 임박, 자동보충 대기 중"

**메트릭 카드 4개**
| 카드 | 내용 |
|------|------|
| 거래처 이상 | 위험 + 경고 거래처 수, 위험/경고 구분 dot |
| 오늘 배송 | 배송 건수, 전일 대비 변화율 |
| 차량 가동 | 운행/전체 대수, 벌크/실린더 구분 표시 |
| 정비 예정 | 이번 주 정비 예정 차량 수 |

**실시간 지도**
- Leaflet 지도 (울산 중심, CartoDB Positron 타일)
- 트럭 마커: 상태별 색상 (운행=초록, 경고=빨강, 대기=회색), 운행 중 펄스 애니메이션
- 배송 경로: 지나온 경로(실선) + 남은 경로(점선), 기사별 색상 구분
- 거래처 마커: 위험도별 색상 (위험=빨강, 경고=노랑, 정상=회색), 표시/숨김 토글
- 정류장 마커: 배송 완료=색 채운 원, 배송 예정=흰색 테두리 원
- 하단 범례: 차량 상태 + 기사별 경로 필터 + 실선/점선/완료/예정 구분

**알림 피드**
- 최근 10건 시간순 내림차순
- 각 알림: 상태 dot + 메시지 + 시간 (카드형 배경)

**배송량 추이 차트**
- 30일 Recharts AreaChart
- 품목별 스택 (N2/O2/CO2/AR/LPG), 품목 색상 코드 적용
- 범례 상단 우측 배치

### 2. 텔레매틱스 (`/telematics`)

차량별 실시간 텔레매틱스 모니터링 센터.

**좌측 패널: 차량 목록**
- 10대 벌크 차량 리스트
- 정렬 기능: 상태 / 건강점수 / 속도 / 기사명 (클릭 시 오름차순/내림차순 토글)
- 정렬 시 layout 애니메이션으로 부드러운 위치 전환
- 각 행: 상태 dot + 차량번호 + 기사명 + 속도(운행) 또는 상태 텍스트 + 미니 건강 게이지

**우측 패널: 선택 차량 상세**

| 섹션 | 내용 |
|------|------|
| **차량 헤더** | 차량번호, 기사, 모델, 품목, 상태 뱃지, 현재 속도(대형 text-4xl), 위치 미니맵 (클릭 → 확대 모달) |
| **배송 미션 현황** | 본사→정류장1(완료)→정류장2(이동중)→정류장3(대기)→본사 타임라인, 적재량/배송량/잔여량 프로그레스 바 |
| **차량 상태** | 반원형 SVG 게이지 3개 (엔진 RPM, 냉각수 온도, 유압) + 배터리 전압 + 연료 잔량 게이지 + 타이어 공기압 4륜 표시 |
| **운전 행동 분석** | 운전 점수(대형 숫자, 색상 코드) + 일일 통계 (주행거리, 평균속도, 공회전) + 안전 이벤트 횟수 + 속도 이력 AreaChart |
| **화물 + 연료** | 적재 중량 게이지 + 탱크 압력/온도 + 연비 + 일일 연료 사용량 + CO2 배출 + 24시간 연료 AreaChart |
| **이벤트 로그** | 최근 이벤트 (시간, 심각도 dot, 유형 뱃지, 설명) + DTC 고장코드 |

**실시간 수치 변동**
- 운행 중 차량: 2초마다 속도, RPM, 온도, 유압, 연비 등 수치가 미세하게 변동
- 연료: 시간 경과에 따라 감소만 (절대 증가 안 함)
- 대기/경고 차량: 정적 수치 유지

**확대 지도 모달**
- 미니맵 클릭 시 85vw×80vh 모달
- 지나온 경로(초록 실선) + 남은 경로(파란 점선) + 정류장 마커 + 본사 마커

### 3. 운행 이력 (`/history`)

차량별 일자별 운행 경로 및 이벤트 조회.

**좌측 패널**
- 차량 선택 리스트 (10대)
- 날짜 선택 (date input)
- 일일 요약: 총 주행거리, 운행시간, 배송 건수, 최고속도

**경로 지도**
- 속도별 색상 경로: 초록(0-40km/h) / 노랑(40-60km/h) / 빨강(60+km/h)
- 번호 매긴 배송 정류장 마커
- 출발/복귀 본사 마커
- 하단 속도 범례

**타임라인 이벤트 로그**
- 25건+ 이벤트 테이블
- 컬럼: 시간, 심각도 dot, 이벤트명, 위치, 속도
- 이벤트 유형: 출발, 도착, 과속, 급제동, 급가속, 공회전, 복귀

**속도 분석 차트**
- AreaChart (07:00~17:00)
- 40km/h, 60km/h 기준선
- 하단 요약: 평균속도, 최고속도, 급제동 횟수, 과속 횟수, 공회전 시간

### 4. 안전 관제 센터 (`/safety`)

위험물 운송 안전 통합 관제 화면.

**안전 등급 배너**
- A/B/C 등급 대형 뱃지 (사고 0건=A, 경고만=B, 사고 있음=C)
- 4개 핵심 수치: 운행 차량, 안전 이벤트, 위험물 적재, 과속 차량
- 최근 위험 알림 카드

**안전 지도**
- 전 차량(20대) 위치 표시
- 과속 차량: 빨간 펄스 마커
- 위험 구역 2곳: 학교 밀집 구역, 주거 밀집 구역 (반투명 빨간 원)
- 차량 툴팁: 차량번호, 기사, 속도, 적재물, 안전점수

**실시간 이벤트 피드**
- 초록 라이브 dot 표시
- 3초마다 새 이벤트 자동 추가
- 8종 이벤트: 과속, 급제동, 급가속, 졸음감지, 위험구역진입, 탱크압력이상, 사각지대경고, 안전복귀
- 위험 이벤트: 빨간 배경 하이라이트

**기사 안전 랭킹**
- 10명 테이블: 순위, 기사명, 차량번호, 안전점수, 급제동, 과속, 사각지대
- 정렬 가능 (점수, 급제동, 과속, 사각지대)
- TOP 3: 금/은/동 메달 아이콘
- 기사 클릭 → 어라운드뷰 연동

**어라운드뷰 모니터링**
- SVG 탱크로리 상면도 (400×400)
- 4방향 카메라 + 감지 영역
- 2.5초마다 객체 이동 시뮬레이션 (보행자/차량/장애물)
- 감지 레벨별 색상 변화 (안전→주의→위험→긴급)
- 일시정지/재생 토글
- 선택된 기사의 차량으로 연동

**안전 통계 3종**
| 차트 | 내용 |
|------|------|
| 일별 이벤트 추이 | 14일 스택 BarChart (과속/급제동/기타) |
| 이벤트 유형 분포 | 6개 카테고리 도넛 PieChart |
| 월간 안전점수 추이 | 6개월 LineChart (70→85 상승, 목표선 80) |

### 5. 수요예측 VMI (`/forecast`)

거래처별 탱크 잔량 예측 및 VMI 자동보충 주문.

**좌측 패널: 거래처 리스트**
- 15개 거래처, 위험도순 정렬 (위험→경고→안전)
- 위험/경고 건수 뱃지
- 검색 필터 (거래처명/품목)
- 각 행: 상태 dot + 거래처명 + 품목 dot + 잔량 게이지 + 고갈 예상일

**우측 패널: 예측 상세**

| 요소 | 내용 |
|------|------|
| 거래처 정보 | 이름, 품목, 탱크 용량, 월 배송 횟수, brand 좌측 accent bar |
| 14일 예측 차트 | 파란 실선(실측 30일) + 주황 실선(VMI 보충 시 예측) + 빨간 점선(보충 없음 시나리오) + 주황 반투명 영역(신뢰구간) |
| 과거 보충 이벤트 | 실측 라인에서 보충 시점 마커 + "기사명 N,NNNkg" 라벨 (우측 상단 배치) |
| 미래 보충 예측 | 초록 세로 점선 + "VMI 보충 N,NNNkg" 라벨 + 초록 원 마커 |
| 오늘 기준선 | 회색 실선 + "오늘" 라벨로 과거/미래 구분 |
| 보충 임계선 | 빨간 점선 20% |
| 요약 카드 4개 | 현재 잔량, 일평균 소비, 고갈 예상일, 권장 보충량 |
| AI 영향 요인 | 상위 3개 요인 (예: "기온 하강 +12%", "가동률 증가 +8%") |
| VMI 주문 버튼 | "VMI 자동보충 주문 생성" → 성공 토스트 |

**커스텀 툴팁**
- 과거: 실측 잔량 + 보충 시 기사/차량/도착시간/보충량/잔량 변화 상세
- 미래: VMI 보충 시/보충 없음 비교 + 보충 이벤트 상세

### 6. 배차 최적화 (`/dispatch`)

AI 기반 최적 배차 경로 계산.

**좌측 패널: 배송 요청 리스트**
- 8건 배송 요청 (VMI 자동 4건 + 카톡 수동 4건)
- **우선순위 시스템**: 긴급(빨강)/높음(노랑)/보통(회색) — 좌측 3px 보더 + 배경색
- **드래그 재정렬**: Framer Motion Reorder로 순서 변경
- **우선순위 클릭 변경**: 뱃지 클릭 시 긴급→높음→보통 순환
- 각 행: 드래그 핸들 + 체크박스 + 거래처명 + 품목 dot + 수량 + 미니 탱크 게이지 + 고갈일 + 우선순위 뱃지 + VMI/카톡 뱃지
- **펼치기 상세**: 제품, 수량, 탱크 잔량, 요청 시간 + 메모(경고 배경)
- 상단 요약: "긴급 2 / 높음 3" 뱃지

**AI 배차 실행 전: 기존 시스템 비교**
- 기존 ERP 배차 화면 스크린샷
- 수기 거래내역장 사진
- 문제점 3개 카드: 배차 40분+, 공차율 7.6%, 긴급배송 11%

**AI 배차 최적화 실행**
- 5단계 프로그레스 (배송 데이터 분석 → 차량 확인 → OR-Tools 경로 최적화 → 차량 매칭 → 검증)
- 선택된 건수에 맞게 동적 텍스트
- BrainCircuit 아이콘 펄스 애니메이션

**최적화 결과**
- 비교 통계 3개: 총 주행거리 (▼N%), 배송 건수/차량 수, 배차 시간 (▼88%)
- 기사별 결과 카드: 차량번호 + 기사 + 방문 순서 + 거리 + 예상 시간
- 카드 클릭 → 해당 기사 경로만 지도에 하이라이트 (나머지 흐리게)
- **상세 타임라인**: 카드 펼치면 본사→A→B→본사 정류장별 배송량 + ETA 표시

**Before/After 지도**
- 탭 전환: Before(빨강, 비효율) / After(초록, 최적화)
- 경로선 + 정류장 마커 + 본사 마커
- 하단 기사별 색상 범례 (클릭 필터)

### 7. 카카오톡 발주 AI (`/kakao`)

자연어 카카오톡 메시지에서 발주 정보를 AI로 자동 파싱.

**좌측: 카카오톡 채팅 UI**
- 카카오톡 스타일 (#B2C7D9 배경, 흰색 말풍선)
- 5건 프리셋 메시지 (예: "질소 3톤 급합니다 오늘중")
- 메시지 클릭 → 우측 파싱 결과 표시
- 직접 입력 + "분석" 버튼
- 파싱 로직: 품목 매칭(산소→O2), 수량 추출(정규식), 긴급도 감지

**우측: AI 파싱 결과**
- 7개 필드 카드: 거래처, 거래처코드, 제품코드, 제품명, 수량, 납기, 긴급도
- 좌측 accent bar: 성공=초록, 주의(신뢰도<80%)=노랑
- 신뢰도 뱃지

**ERP 주문 생성 모달**
| 영역 | 내용 |
|------|------|
| 주문번호 | ORD-YYYYMMDD-NNN 자동 생성 |
| 주문 정보 | 거래처, 코드, 제품(품목 dot), 수량, 납기, 긴급도 — 2열 그리드 |
| AI 신뢰도 | 프로그레스 바 시각화 |
| 배차 연동 | "배차최적화에 자동 반영" 체크박스 |
| 원본 정보 | 발주 채널, 원본 메시지, 처리 담당 |
| 하단 버튼 | 취소 / 인쇄 / 주문 확정 → |
| 확정 후 | 체크 애니메이션 + 3단계 완료 표시 + "배차최적화로 이동" 버튼 |

**하단 통계 바**
- 오늘 카톡 발주 12건, 자동 인식 11건 (91.7%), 평균 처리 18초

### 10. 성과 KPI (`/kpi`)

AI 도입 성과 대시보드.

**종합점수 배너**
- 네이비 그래디언트 배경
- 원형 프로그레스 링 (100점 만점, 애니메이션)
- "AI 도입 성과 대시보드" 타이틀
- 미니 수치 3개: 개선 지표 6개, 물류비 1.5억 절감, CO2 120톤 감축

**디지털 전환 Before/After**
- 3열 비교: 종이 장부 시대 (실물 장부 사진 + 문제점 X 아이콘) → AI 전환 화살표 → AI 통합 관제 (이점 체크 아이콘)

**6개 KPI 메트릭 카드** (3열 그리드)
| 지표 | Before | After | 개선 |
|------|--------|-------|------|
| 납기단축율 | 기준 | 30%↑ | 75점 |
| 공차율 | 7.6% | 3.2% | 85점 |
| 긴급배송 | 11.0% | 2.2% | 80점 |
| 운송사고 | 기준 | 60%↓ | 90점 |
| 물류비 | 기준 | 1.5억↓ | 70점 |
| CO2 배출 | 기준 | 120톤↓ | 65점 |

- 각 카드: 원형 프로그레스 링(아이콘 내장) + 좌측 컬러 accent bar + Before→After 카운트업 애니메이션

**시스템 통합 플로우 다이어그램**
- 7단계: 발주접수 → AI파싱 → ERP등록 → 수요예측 → 배차최적화 → 배송 → 납품확인
- AI 단계: brand 색상 강조

**월별 성과 추이 차트**
- ComposedChart (Line + Area 그래디언트)
- 4개 라인: 납기단축율, 공차율, 긴급배송, 물류비
- 토글 pill 버튼 (활성=컬러, 비활성=회색)
- 마일스톤: 3월 "파일럿", 6월 "전면확대"

## 발표 시 핵심 시연 포인트

1. **수요예측**: 거래처 클릭 → 14일 예측 차트 (보충/미보충 비교) → "자동보충" 버튼 → 토스트
2. **배차**: 우선순위 드래그 → "AI 배차 실행" → 5단계 프로그레스 → Before/After 지도 경로 → 기사별 상세
3. **카톡 AI**: 메시지 클릭 → 파싱 결과 → ERP 주문 모달 → 주문 확정 → 배차 연동
4. **텔레매틱스**: 차량 선택 → 실시간 수치 변동 → 배송 미션 현황 → 지도 확대
5. **안전 관제**: 실시간 이벤트 피드 (3초마다 추가) → 위험 구역 지도 → 기사 랭킹 → 어라운드뷰
6. **예지보전**: 히트맵 클릭 → RUL 게이지 애니메이션 → 정비 작업지시서 인쇄
7. **KPI**: 종합점수 카운트업 → 종이장부 비교 → 6개 지표 + 월별 추이
